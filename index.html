Recent Projects

Project 1

Human Action Recognition

In this project, our focus was on Human Action Recognition using advanced image classification techniques. Leveraging state-of-the-art machine learning algorithms, we developed a predictive model that accurately identifies and classifies human actions from images. Through the integration of deep learning and computer vision methodologies, our system demonstrated a high level of precision in recognizing diverse human activities. From basic gestures to complex movements, our model excels in capturing the nuances of human actions, making it a powerful tool for applications in surveillance, gesture-based interfaces, and human-computer interaction.

Project 2

NYC School Data Cleaning & Analysis

TIn this project, I conducted a comprehensive data cleaning and analysis initiative focused on NYC school data. To gain deeper insights, I employed Partial Differential Equations (PDE) and visualization techniques, unveiling intricate correlations within the data. This thorough analysis not only provided a clearer understanding of the dataset's structure but also laid the foundation for future studies and informed decision-making in the realm of education.

Project 4

Academic Project - "Tweet Store, Search, and Cache"

In this academic endeavor, we delved into the intricate world of modern data management systems with a focus on the development of the "Tweet Store, Search, and Cache" system. The project involved storing extensive JSON records with up to 80 key/value fields in both relational and non-relational databases, namely MongoDB and MySQL. Through the implementation of a search engine using Flask, we facilitated user-friendly data storage and retrieval. The integration of caching mechanisms not only enhanced search performance but also minimized disk reads, significantly improving system latency. This project showcased a deep understanding of database systems, effective use of Python programming, and the creation of a robust search application with real-world applicability in handling large-scale, unstructured datasets.
